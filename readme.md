# NodeInfo
NodeInfo使用NodeQuery API提供的数据，提供简易的服务器监控信息，可以很方便的放到自己网站，供管理员参阅查看。

### 环境要求
* PHP

### 使用说明
* 下载源码后放到站点目录下，修改`ListServers.php`填写NodeQuery API即可。
* 详细说明请参考《[Linux服务器监控工具NodeQuery与NodeInfo](https://www.xiaoz.me/archives/10153)》
